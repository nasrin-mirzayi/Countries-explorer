# Countries Explorer

A React app that fetches country data from the REST Countries API.

## Features
- Search by country name
- Filter by region
- Loading and error handling

## API Used
https://restcountries.com/

## How to Run
search a country name clicking on input.
or you can filter by region.
each country consists of: flag image, country name, region and population.
